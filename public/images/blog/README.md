# Blog Images Directory

This directory contains images for blog posts.

## Current Images:
- `blog-nature.jpg` - Nature/Art themed blog image
- `blog-design.jpg` - Design/Marketing themed blog image  
- `blog-tech.jpg` - Technology themed blog image
- `avatar-1.jpg` - Author avatar image

## Image Requirements:
- **Format**: JPG or PNG
- **Resolution**: 800x600px or higher
- **File Size**: Optimize for web (under 500KB each)
- **Aspect Ratio**: 4:3 or 16:9 recommended

## Recommended Content:
- High-quality, relevant images for each blog category
- Professional author avatars
- Images that match the blog post themes
- Optimized for web performance

## Fallback:
If images are missing, the blog cards will show placeholder content.
