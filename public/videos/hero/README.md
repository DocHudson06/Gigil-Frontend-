# Hero Video Files

This directory should contain the following video files for the hero section:

## Required Files:
- `hero.mp4` - Primary video format (MP4)
- `hero.webm` - Fallback video format (WebM)
- `poster.jpg` - Video poster image (optional)

## Video Requirements:
- **Format**: MP4 (primary) and WebM (fallback)
- **Resolution**: 1920x1080 or higher
- **Duration**: 10-30 seconds (loops automatically)
- **File Size**: Optimize for web (under 10MB)
- **Content**: Should be relevant to your brand/gifts theme

## Recommended Content Ideas:
- Gift wrapping process
- People receiving gifts
- Product showcase
- Lifestyle scenes
- Abstract patterns related to gifting

## Optimization Tips:
- Use video compression tools like HandBrake
- Consider using online tools like CloudConvert
- Test on different devices for performance
- Ensure good contrast for text overlay readability

## Fallback:
If no video files are provided, the hero section will show a solid background color.
